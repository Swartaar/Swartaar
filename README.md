# SwarTaar — India’s Guitar Learning Hub 🎸

A free, single-file web app for learning guitar the Indian way — Bollywood, Hindi,
Gujarati and Sufi songs, Sargam/raga training, an interactive fretboard, a built-in
tuner, and a practice dashboard.

**Live site:** *add your GitHub Pages link here once deployed*

## Features

- Beginner → advanced lesson path with progress you can check off
- Sargam & raga training mapped onto the fretboard
- 40+ songs with lyrics, chords, tabs and capo helper
- Chord library with playable audio (multiple guitar tones)
- Reference + live-microphone tuner
- Practice dashboard: timer, daily streaks, chord-change speed drill
- **Accounts** — sign up / log in, with per-user streaks, saved songs and badges
  (stored in the browser; no server needed)

## Run it

It’s one self-contained file. Just open `index.html` in any modern browser —
nothing to install.

## Deploy for free (GitHub Pages)

1. Put `index.html` in a GitHub repository.
1. Repo **Settings → Pages**.
1. Under **Branch**, choose `main` and `/ (root)`, then **Save**.
1. Wait ~1 minute — your site goes live at
   `https://<your-username>.github.io/<repo-name>/`

## Note on accounts

Sign-ups and progress are saved locally in each visitor’s browser (per device).
To sync across devices later, the code includes a commented Firebase upgrade
path near the bottom of `index.html`.
